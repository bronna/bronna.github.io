Building a Parametric Shell

  • WebGL
  • regl
  • Three.js
  • Shaders
  • Observable
  • Javascript
  • 3D

Like many humans, I’m awestruck by the beautiful geometries found in nature. Something like a logarithmic spiral can be found in places as vast as a galaxy, and as small as a seashell. Wondering how math can model some of these wonders, I stumbled upon a paper by a professor in Portugal. Step by step, the professor outlined the mathematics of shell structures. Excitedly, I learned that one equation can model nearly all seashells on Earth.

Using Javascript and Three.js, I made progressive models that go through each step of the equation. Eventually, the reader comes to a sandbox (hehe) where they can experiment with the parameters of the seashell equation to find new, familiar, and fun structures.

Using Three.js, I ran up against performance constraints because of the number of WebGL instances on one page. On Twitter, software experts suggested I try an Observable notebook, which got me halfway there. Next, they recommended I port the code to regl, a lower-level WebGL library. With their guidance to get started, I ended up diving into the world of shaders and rotation matrices. I hadn't anticipated getting so deep into the math, but after dusting off old textbooks, I was able to successfully create multiple interactive instances on a page.

This project solidified my love for creating explorative tools. The generator resonated with many people, which was a fulfilling way to connect with fellow nerds during the COVID pandemic. I also enjoyed re-visiting my love of 3D, which started during my days doing CAD as an Apple engineer.

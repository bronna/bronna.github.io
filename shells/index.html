Building a Parametric Shell

  • WebGL
  • regl
  • Three.js
  • Shaders
  • Observable
  • Javascript
  • 3D

Like many humans, I’m awestruck by the beautiful geometries found in nature. Something like a logarithmic spiral has echoes in places as vast as a galaxy, and as small as a seashell. Wondering about the math that can model these miracles, I found a paper by a professor in Portugal. Step by step, the author outlines the mathematics of shell structures. It turns out, one equation can model nearly all of the seashells on Earth.

Using Javascript and Three.js, I made a series of progressive models that walk through each step of the equation. Eventually, I wanted the reader to come to a sandbox (hehe) in which they could experiment with the parameters of the seashell equation to find new, and familiar, structures.

Using Three.js, I ran up against performance constraints because of the number of WebGL instances on one page. Over Twitter, some software experts suggested I try an Observable notebook, which got me halfway there. Next, they recommended I port the code to regl, a lower-level WebGL library. With some help getting started, I ended up diving into the world of shaders and rotation matrices. I hadn't anticipated getting so deep into math, but after dusting off old college textbooks, I was able to create multiple interactive instances on a page.

Finishing this project solidified my love for creating tools that enable exploration. Finding insights in data is wonderful, but empowering others to explore has its own satisfaction. The explainer resonated with many people, which was a rewarding way to connect with others in the midst of the COVID pandemic. I was also able to return to my love of 3D, which started during my days doing CAD as an engineer at Apple.
